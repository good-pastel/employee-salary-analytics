Tentang Project

Dibangun dari Awal dengan Python

Project ini dibangun untuk menunjukkan implementasi struktur data Linked List dalam Python tanpa bergantung pada library analisis data seperti Pandas. Seluruh fitur dikembangkan dari awal, mulai dari operasi CRUD, pencarian data, Filtering, Grouping, hingga perhitungan statistik pada dataset gaji karyawan.

Python Linked List Built From Scratch Object Oriented Statistics Filter CRUD

Python Linked List

Mengimplementasikan struktur data Linked List dari awal dengan menggunakan Node yang saling terhubung, tanpa menjadikan list bawaan Python sebagai media penyimpanan utama.

Object Oriented

Dikembangkan dengan prinsip Object-Oriented Programming (OOP) melalui pemisahan class Employee, Node, dan LinkedList agar kode lebih modular dan mudah dikelola.

CRUD Operations

Mendukung operasi Create, Read, Update, Delete, pencarian data, serta pembaruan data secara batch.

Statistics

Menghitung berbagai statistik seperti total gaji, rata-rata, gaji tertinggi, gaji terendah, pengelompokan data, dan distribusi gaji.

Filter

Mendukung dynamic filtering berdasarkan berbagai atribut karyawan menggunakan fungsi getattr() pada Python.

Built From Scratch

Seluruh fitur dan algoritma dikembangkan dari awal untuk menerapkan konsep struktur data, algoritma, dan OOP tanpa menggunakan library analisis data seperti Pandas.

Menyediakan operasi penyaringan data secara dinamis berdasarkan atribut karyawan serta mendukung pembaruan data secara batch dengan tetap mempertahankan struktur Linked List.

Signature filter_by(attribute, value)

Complexity O(n)

Methods

  • filter_by(attribute, value)

Dokumentasi

Dokumentasi Employee Salary Analytics.

Employee Salary Analytics merupakan library berbasis Python yang mengimplementasikan struktur data Linked List untuk mengelola dan menganalisis data gaji karyawan. Library ini menyediakan operasi CRUD, Filtering, Grouping, Statistics, serta Salary Distribution tanpa bergantung pada library analisis data seperti Pandas.

Installation

Clone repository, kemudian jalankan aplikasi menggunakan Python. Project ini dibangun menggunakan Python Standard Library sehingga tidak memerlukan dependensi pihak ketiga.

terminal 
git clone https://github.com/good-pastel/employee-salary-analytics

cd employee-salary-analytics

python main.py

Quick Start

Muat dataset CSV ke dalam implementasi Linked List, kemudian gunakan Method yang tersedia untuk melakukan operasi CRUD, Filtering, Grouping, serta analisis data gaji.

main.py 
employees = load_csv()

ll = LinkedList()

for employee in employees:
    ll.append(employee)

print(ll.average_salary())
