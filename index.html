About This Project

Built From Scratch Using Python

Employee Salary Analytics is a custom analytics library developed entirely in Python using a manually implemented Linked List data structure. The project demonstrates object-oriented programming, CRUD operations, statistical analysis, grouping, filtering, and clean architecture without relying on external data processing libraries.

Python Linked List Built From Scratch Object Oriented Statistics API Filter API CRUD

Python Linked List

A custom Linked List implementation created from scratch without using Python's built-in list as the main storage structure.

Object Oriented

The project follows object-oriented principles by separating Employee, Node, and LinkedList into independent classes.

CRUD Operations

Supports Create, Read, Update, Delete, Search, and Batch Update operations.

Statistics API

Calculate totals, averages, highest salary, lowest salary, grouping and salary distribution.

Filter API

Dynamic filtering by any employee attribute using Python's getattr().

Built From Scratch

Designed for learning data structures, algorithms and software architecture.

Project Architecture

How The Application Works

This project follows a layered architecture. Data starts from the CSV dataset, is converted into Employee objects, stored inside a custom Linked List, processed by multiple APIs, and finally displayed through the application.

ds_salaries.csv

📥 Dataset

Stores the employee salary dataset used by the application.

csv_loader.py

📥 Data Import

Reads the CSV dataset and converts every row into an Employee object.

employee.py

🏷 Data Model

Represents a single employee record with salary, job title, work year, experience level, and other attributes.

node.py

🔗 Linked List Node

Wraps every Employee object into a node that can be connected inside the Linked List.

linked_list.py

⭐ Core Library

Acts as the core of the application by managing the custom Linked List and providing all business logic.

CRUD Statistics Grouping Filter Distribution

main.py

🚀 Entry Point

Loads the dataset, calls the APIs, and displays the analytics results.

Employee Salary Analytics

📊 Output

Presents salary insights generated from the custom Linked List.

The Employee Salary Analytics Library provides a collection of reusable APIs built on top of a custom Linked List implementation. Each API focuses on solving common employee analytics tasks while demonstrating object-oriented programming and clean software design.

APIs that can be called directly by applications using the Employee Salary Analytics Library.

Filter API

Query

Provides flexible filtering and bulk update operations using dynamic attributes while preserving the original Linked List structure whenever possible.

Signature filter_by(attribute, value)
Returns LinkedList
Complexity O(n)

Methods

  • filter_by(attribute, value)
  • update_all_by(attribute, old_value, new_value)
  • remove_all_by(attribute, value)

Developer Notes

Returns a new LinkedList instead of modifying the original list, allowing additional operations to be chained safely.

filter.py 
filtered = ll.filter_by(
    "experience_level",
    "SE"
)

filtered.display()

Documentation

Employee Salary Analytics Documentation

Employee Salary Analytics is a custom Python library that demonstrates how real-world employee salary data can be managed and analyzed using a manually implemented Linked List. The project includes CRUD operations, filtering, grouping, statistical analysis, and salary distribution without relying on external data processing libraries such as Pandas.

Installation

Clone the repository and run the application locally using Python. The project is built entirely with the Python Standard Library and does not require any third-party dependencies.

terminal 
git clone https://github.com/good-pastel/employee-salary-analytics

cd employee-salary-analytics

python main.py

Quick Start

Load employee records from a CSV dataset into a custom Linked List implementation, then perform CRUD operations, filtering, grouping, and salary analytics using the available APIs.

main.py 
employees = load_csv()

ll = LinkedList()

for employee in employees:
    ll.append(employee)

print(ll.average_salary())

Environment

Built entirely with the Python standard library, making the project lightweight and easy to run.

  • Python 3.10+
  • CSV Dataset (ds_salaries.csv)
  • Standard Library only
